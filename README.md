# NembusUI

NembusUI is a component library, which is easy to use, customized CSS framework for making projects beautifully and faster.

## Technologies used
- HTML
- CSS

## Components defined in NembusUI
- Alerts
- Avatars
- Badges
- Buttons
- Cards
- Grid
- Image
- Ratings
- Slider
- Textboxes
- Text Utilities
